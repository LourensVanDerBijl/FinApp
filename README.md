# FinApp
FinAPP is a secure, cross‑platform financial management tool designed for couples to fairly split household expenses while keeping personal savings and debts private. The system includes:
Manager Portal: Admins manage client registrations, approve payment options, and oversee subscriptions.
User Website: Customers register, log in, and access their account dashboard to track shared contributions and personal expenses.
Mobile App (Flutter): Companion app for reminders, notifications, and on‑the‑go expense tracking.
Backend: C# APIs with PostgreSQL database for secure data storage.
Authentication: Firebase for login, registration, and client management.
Core Values: Fairness, transparency, privacy, and trust in shared financial planning.
